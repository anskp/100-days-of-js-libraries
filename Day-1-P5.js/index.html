P5.js Creative Gallery

Interactive visualizations created with pure p5.js

About This Application

This application showcases the power of p5.js for creative coding and visualization. All interactions and visualizations are built using p5.js in a structured, object-oriented manner.

Model

Data structures and properties driving our visualizations, including particles, vectors, and noise parameters.

View

The visual rendering of sketches using p5.js's drawing capabilities, organized into separate renderers.

Controller

Logic that connects user interactions to visualization behavior and manages application state.
